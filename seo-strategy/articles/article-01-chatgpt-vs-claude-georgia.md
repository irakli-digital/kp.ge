## ტესტი #1: კრეატიული წერა

**დავალება:** "დაწერე მოკლე მოთხრობა ზაფხულის დღის შესახებ თბილისში"

### ChatGPT (GPT-4) შედეგი:

> თბილისის ცხელ ზაფხულის დღეს, მზის სხივები აბანოთუბნის ქუჩებს ოქროსფრად ღებავდა. ნინო ყავის ფინჯანით ხელში დგას აივანზე და მტკვარს გადმოჰყურებს...

**შეფასება:**
- სტრუქტურა: ⭐⭐⭐⭐⭐
- ქართული: ⭐⭐⭐⭐
- კრეატიულობა: ⭐⭐⭐

### Claude 3.5 შედეგი:

> რუსთაველის გამზირზე სიცხისგან თავშესაფარს ვეძებდი, როცა პატარა ეზოში შევედი. იქ მოხუცი ბებია ლობიოს არჩევდა და ისე მშვიდად მიმზერდა, თითქოს დრო აქ სხვანაირად მიდიოდა...

**შეფასება:**
- სტრუქტურა: ⭐⭐⭐⭐
- ქართული: ⭐⭐⭐⭐⭐
- კრეატიულობა: ⭐⭐⭐⭐⭐

**გამარჯვებული:** Claude 3.5 - უფრო ბუნებრივი, ემოციური ტექსტი.

>

>
> პატივისცემით,
> [სახელი]

### Claude 3.5 შედეგი:

>
> მომწერეთ, როცა მოგეხერხებათ.

**გამარჯვებული:** ორივე კარგია, მაგრამ:
- **GPT-4** - უფრო ფორმალური, კორპორატიული

**ორიგინალი:** "The early bird catches the worm, but the second mouse gets the cheese."

### Claude 3.5:

> "ვინც ადრე ადგება, მოგება მოაქვს - თუმცა ზოგჯერ მოლოდინმაც იცის თავისი."

**გამარჯვებული:** Claude 3.5 - უფრო იდიომატური, აზრის გადმოცემა ნაცვლად პირდაპირი თარგმანისა.

## ტესტი #4: ტექნიკური ტექსტი

**დავალება:** "აგიხსენი რა არის API მარტივი სიტყვებით"

### ChatGPT (GPT-4):

> API (Application Programming Interface) არის კავშირის წერტილი ორ პროგრამას შორის. წარმოიდგინეთ რესტორანი: თქვენ (აპლიკაცია) ვერ შედიხართ სამზარეულოში (სერვერი), მაგრამ ოფიციანტი (API) იღებს თქვენს შეკვეთას და მოაქვს საჭმელი.
>
> **მაგალითი:** როცა აპლიკაცია ამინდს აჩვენებს, ის API-ით "ეკითხება" მეტეოსამსახურის სერვერს მონაცემებს.

### Claude 3.5:

> API არის როგორც თარჯიმანი ორ პროგრამას შორის. ერთი პროგრამა "ეუბნება" მეორეს რა უნდა, API თარგმნის და პასუხს აბრუნებს. ისე, როგორც რესტორანში მენიუ - არ გჭირდება იცოდე როგორ მზადდება კერძი, უბრალოდ ირჩევ რა გინდა.

**გამარჯვებული:** GPT-4 - უფრო სტრუქტურირებული, კონკრეტული მაგალითით.
